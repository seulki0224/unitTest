02. NVM 사용법

NVM을 통한 Node JS 설치 및 버전관리방법
Node JS 
            터미널
            nvm list 확인
            > nvm ls

            NodeJS 버전 확인해서 NVM을 통해 설치
            그냥 설치하면 이미 설치된 NodeJS랑 부딪히므로 NVM사용
            연습할떄는 최신버전 사용 웬만하면 LTS 설치하세요.(안정적인 짝수번호)

            NodeJS 버전 확인
            https://nodejs.org 
            
            nvm 설치
            > nvm install 20.11.1

            최신버전 추가설치
            nvm install 21.6.2

            어떤 버전을 쓸지 선택
            nvm use 20.11.1

            nvm ls
             -> v20.11.1
                v21.6.2

            node JS 버전 확인
                nvm use 20.11.1랑 같이 적용됨
            
            node --version
            v20.11.1

            해당 버전 삭제
            nvm uninstall 12.21.0

            nvm 으로 사용할수 있는 명령 확인
            nvm --help
